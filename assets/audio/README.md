# 音声アセットをここに配置してください

このフォルダに以下の2ファイルを置いてください（ファイル名は完全一致させること）:

- `brown_noise.ogg` … 集中フェーズ用のループ再生用ブラウンノイズ。
  - **OGG(Vorbis) 形式**を使用（MP3 と違いギャップレスなので、`LoopMode.one` でループしても継ぎ目にプツッという音が出ない）。
  - 素材は IFFT で「最初から完全に周期的」に生成しており、先頭と末尾が数学的に連続＝継ぎ目ゼロ。
- `break_noise.ogg` … 休憩フェーズ用の、より柔らかく落ち着いたループ音（同じくギャップレス OGG・周期生成）。
- `countdown.mp3` … フェーズ切り替え 3 秒前に鳴る「3・2・1」カウントダウン（またはアラート）音。約 3 秒（ループしないので MP3 で問題なし）。

生成方法の参考: ブラウン/休憩ノイズは逆フーリエ変換でランダム位相・ブラウン系スペクトルから合成し、`ffmpeg -c:a libvorbis` で OGG 化しています。

配置後、`pubspec.yaml` の `assets:` に `assets/audio/` が含まれていることを確認し、
`flutter pub get` を実行してください。
