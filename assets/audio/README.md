# 音声アセットをここに配置してください

このフォルダに以下の2ファイルを置いてください（ファイル名は完全一致させること）:

- `brown_noise.mp3` … 集中フェーズ用のループ再生用ブラウンノイズ。
  - シームレスにループさせるため、**先頭と末尾が無音・無クリックになっている素材**を推奨します。
  - 数十秒〜数分の長さで問題ありません（`LoopMode.one` で連続再生します）。
- `break_noise.mp3` … 休憩フェーズ用の、より柔らかく落ち着いたループ音。集中用と同様にシームレスループ推奨。
- `countdown.mp3` … フェーズ切り替え 3 秒前に鳴る「3・2・1」カウントダウン（またはアラート）音。約 3 秒。

配置後、`pubspec.yaml` の `assets:` に `assets/audio/` が含まれていることを確認し、
`flutter pub get` を実行してください。
